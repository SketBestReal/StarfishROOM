How to Play

Keyboard & Mouse

WASD — Move
Left Click — Attack
Right Click — Dash
E/RControl — Throw bomb

Gamepad (Xbox Input Only):
Left stick — Move/Select Crads
B — Attack
Y — Dash
X — Throw bomb
